¿QUIENES SOMOS?

Para comenzar...

Somos un grupo de amigos que decidimos realizar una pagina web para la venta de productos de mascotas como por ejemplo: bandanas, correas, identificaciones, etc. Todo esto empezó con la idea de ayudar a una fundación sin fines de lucro. Hoy en día es casi imposible dar abasto a la demanda de productos a través de redes sociales, por ende, construimos esta página web para poder administrar de mejor manera nuestras ventas, presupuestos y el despacho de nuestros productos.
